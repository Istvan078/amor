# Amor ❤️  
A modern dating app focused on clean UX, speed, and real-time communication.

## ✨ Features
- 🔐 Sign up / Sign in
- 👤 Profile creation (photos, bio, interests)
- 🎯 Matching logic (filters & preferences) // Still under dev
- ❤️ Like / Match system // Still under dev
- 💬 Real-time chat (messages, read states / statuses)
- 🔔 Notifications ( push-ready ) // Still under dev
- 🧭 Connections & friend-like interactions // Still under dev
- ⚡ UX & performance-first UI (responsive, fast, minimal clutter)

> Goal: build a *real, usable* dating product with a scalable architecture and a smooth experience on web and mobile.

## 🧱 Tech Stack
- **Frontend:** Angular  
- **State management:** NgRx
- **Styling:** SCSS

## 🚀 Getting Started

### 1) Install dependencies
run command in terminal: npm install
